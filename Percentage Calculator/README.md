# Percentage Calculator

## Purpose
The **Percentage Calculator** is a responsive and interactive frontend tool designed to calculate the percentage of marks obtained. It provides instant visual feedback using a progress bar, making it ideal for educational or quiz applications.

## Key Features
- **User Inputs:** 
  - Obtained marks
  - Total marks  
- **Validation Rules:** 
  - Ensures input values are numbers
  - Prevents division by zero
  - Handles empty or invalid inputs gracefully
- **Calculation Formula:**  
  \[
  \text{Percentage} = \frac{\text{Obtained Marks}}{\text{Total Marks}} \times 100
  \]
